# Markdown-server

Markdown-server 提供了Markdown的数学公式 `Lexte`，以及流程图`yUML`服务端渲染支持。

## 如何使用

- clone 本项目
    - `git clone https://github.com/liuxy0551/markdown-server`
- 安装依赖
    - `npm install` 或 `yarn`
- 启动服务
    - `node index.js`
